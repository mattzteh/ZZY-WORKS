# README

[ZZY WORKS](https://zzyworks.herokuapp.com/) is a clone of RAMA WORKS, an e-commerce keyboard company based in Australia. <br />
This app used Ruby on Rails as its backend, and JavaScript, Redux + ReactJs as its frontend.

<h2>Front Page</h2>

<h3>Features</h3>

* Log In/Log Out
* Create an account
* Demo User with full functionality
* Search Bar to look for specific products
* Add/remove from cart that keeps track of quantity per product (can increment/decrement quantity within cart):
